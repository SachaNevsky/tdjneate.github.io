About Inker

Inker is a prototype tool for people with aphasia to create constrained visual art. This version is for demonstration purposes for the ASSETS 2020 conference, where the work is presented as a poster. The paper related to the poster, 'Painting a Picture of Accessible Digital Art' is available via the ACM and via City's Open Access portal. A video summarising Inker is embedded below.

Explainer and Usage Instructions

Inker uses the colour information from a selected source image to support visual creativity. The drawing canvas is overlaid on the source image. When using Inker, a user draws by pressing down a finger or stylus on the drawing canvas. The colour of the brush is then matched to the colour at the same position in the underlying image. Drawing with this colour is then done in one of two modes:
  • Inking uses a standard drawing algorithm, with a circular brush. Brush thickness is controlled with a slider
  • Rubbing enables the user to place shapes of varying sizes around the canvas. The user selects brush shape (circles or squares) and uses sliders to control the size of the brush (Rubbing Shape Size) and the speed at which shapes are placed (Rubbing Speed). This effect is analogous to spray painting with specific shapes, whose colour is determined by the original artwork.
